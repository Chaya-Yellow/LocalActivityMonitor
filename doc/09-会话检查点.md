```
【开发前】
  1. 读本文档 → 知道当前进度、上次中断位置
  2. 读 doc/07-运行报错日志.md → 避免重复踩坑
  3. 读 doc/04-拆分任务.md 对应模块的任务清单

【开发中】
  4. 每完成一个最小任务，更新本文档"任务完成情况"
  5. 遇到关键决策，记入"决策记录"
  6. 遇到报错，记入 doc/07-运行报错日志.md

【会话结束/中断时】
  7. 更新本文档全部状态：哪些做了、哪些没做、下一步是什么
```
