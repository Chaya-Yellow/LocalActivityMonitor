# 活动监控器 — 测试用例

> 创建日期：2026-07-21
> 测试框架：xUnit + FluentAssertions
> 数据库：SQLite In-Memory（数据层测试）
> Mock 策略：NSubstitute（接口 Mock）

---

## 一、测试分层策略

```
┌─────────────────────────────────────────┐
│           E5 性能与集成测试              │  ← 端到端、长时间运行
├─────────────────────────────────────────┤
│    E4 统计与报表测试   │  E3 内容解析测试 │  ← 纯逻辑、高覆盖
├───────────────────────┴─────────────────┤
│           E2 追踪引擎测试               │  ← 状态机 + Mock
├─────────────────────────────────────────┤
│           E1 数据层测试                 │  ← 内存 SQLite
└─────────────────────────────────────────┘
```

| 层级 | 覆盖目标 | 运行频率 |
|------|---------|---------|
| E1 数据层 | ≥ 90% | 每次提交 |
| E2 追踪引擎 | ≥ 85% | 每次提交 |
| E3 内容解析 | ≥ 95% | 每次提交 |
| E4 统计报表 | ≥ 90% | 每次提交 |
| E5 性能基准 | N/A（指标达标） | 每日一次 |
